# Отчет о проделанной работе - Librarium Frontend

## Краткое описание

Выполнена полная реорганизация и обновление фронтенд-части сайта книжной библиотеки Librarium. Приложение получило современный, премиальный дизайн с glassmorphism эффектами, плавными анимациями и полной адаптивностью для всех устройств.

---

## 1. Реорганизация структуры файлов

### Новая модульная архитектура:

```
src/
├── components/
│   ├── common/          # Общие компоненты (Button, Card, Input, Modal)
│   ├── layout/          # Компоненты макета (Header, Footer)
│   ├── cards/           # Компоненты карточек (BookCard, AudioBookCard)
│   └── carousels/       # Компоненты каруселей (BookCarousel, AudioBookCarousel)
├── pages/
│   ├── auth/            # Страницы авторизации (Login, Register)
│   ├── ProfilePage/     # Страница профиля
│   ├── HomePage.jsx     # Главная страница
│   └── StatsPage.jsx    # Страница статистики
├── hooks/               # Пользовательские хуки
├── services/            # Сервисы для API запросов
├── utils/               # Утилиты и хелперы
└── styles/              # Стили
    ├── globals.css      # Глобальные стили
    └── pages/           # Стили страниц
```

### Преимущества новой структуры:
- ✅ Модульность и переиспользуемость компонентов
- ✅ Удобный импорт через индексные файлы
- ✅ Разделение ответственности
- ✅ Легкость масштабирования и поддержки

## 2. Премиальный дизайн

### Цветовая палитра:
- **Основной цвет:** Оранжевый (#FF6B35, #E85D04)
- **Фон:** Черный (#000000, #111111)
- **Серые оттенки:** #2C2C2C, #4A4A4A, #F5F5F5
- **Текст:** Белый (#FFFFFF), серый (#C5C5C5)

### Шрифты:
- **Основной:** Inter (для текста)
- **Заголовки:** Playfair Display (для заголовков)

### Эффекты:
- **Glassmorphism:** Размытие фона, полупрозрачные карточки
- **Тени:** Мягкие тени для глубины
- **Градиенты:** Оранжевые градиенты для акцентов
- **Свечение:** Оранжевое свечение при наведении

## 5. Работа с бэкендом

### API сервисы:
- **authApi** - авторизация, регистрация, выход
- **profileApi** - данные профиля
- **booksApi** - книги, категории, поиск

### Endpoints:
- `POST /auth/login` - вход
- `POST /auth/register` - регистрация
- `POST /auth/logout` - выход
- `GET /profile` - профиль пользователя
- `GET /books/category` - книги по категории
- `GET /books/featured` - рекомендуемые книги
- `GET /books/search` - поиск книг

### Обработка ошибок:
- ✅ Спиннеры загрузки
- ✅ Toast-уведомления
- ✅ Fallback UI при ошибках
- ✅ Обработка 401 (неавторизован)

## 7. Созданные компоненты

### Общие компоненты (common):
1. **Button** - универсальная кнопка с вариантами (primary, secondary, outline, ghost, danger)
2. **Card** - карточка с glassmorphism эффектом
3. **Input** - поле ввода с валидацией
4. **Modal** - модальное окно с анимациями

### Компоненты макета (layout):
1. **Header** - шапка с навигацией, поиском, меню пользователя
2. **Footer** - подвал с ссылками и соцсетями

### Компоненты карточек (cards):
1. **BookCard** - карточка книги с hover-эффектами
2. **AudioBookCard** - карточка аудиокниги

### Компоненты каруселей (carousels):
1. **BookCarousel** - карусель книг с навигацией
2. **AudioBookCarousel** - карусель аудиокниг

## Инструкция по запуску

### Предварительные требования:
- Node.js 18+
- npm или yarn

### Установка зависимостей:
```bash
cd library-reports
npm install
```

### Запуск в режиме разработки:
```bash
npm run dev
```

### Сборка для продакшена:
```bash
npm run build
```

### Просмотр собранного приложения:
```bash
npm run preview
```

### Запуск через Docker:
```bash
docker-compose up -d
```
