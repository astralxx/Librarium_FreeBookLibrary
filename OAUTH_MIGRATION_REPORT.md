# Отчет о замене OAuth провайдера: Google → Yandex

## Введение

Данный отчет описывает процесс замены OAuth провайдера с Google на Yandex в проекте книжного магазина Librarium. В отчете представлены выполненные изменения, исправленные уязвимости и инструкции по настройке.

## Выполненные задачи

### 1. Замена OAuth провайдера: Google → Yandex

#### 1.1 Обновление auth-service

**Измененные файлы:**
- `auth-service/main.go` - Основной файл с логикой аутентификации
- `auth-service/go.mod` - Обновление зависимостей
- `auth-service/.env.example` - Добавление переменных для Yandex OAuth

**Ключевые изменения:**

1. **Замена JWT библиотеки:**
   - Удалена устаревшая библиотека `dgrijalva/jwt-go`
   - Добавлена современная библиотека `golang-jwt/jwt/v5`

2. **Реализация Yandex OAuth:**
   - Добавлены функции `yandexLoginHandler` и `yandexCallbackHandler`
   - Реализован обмен кода на токен через `https://oauth.yandex.ru/token`
   - Добавлен запрос информации о пользователе через `https://login.yandex.ru/info`

3. **Обработка ошибок:**
   - Добавлена валидация входных данных
   - Обработка ошибок при отсутствии email
   - Логирование ошибок OAuth

#### 1.2 Обновление gateway service

**Измененные файлы:**
- `gateway/main.go` - Обновление маршрутов OAuth

**Ключевые изменения:**
- Маршруты `/auth/google` и `/auth/google/callback` заменены на `/auth/yandex` и `/auth/yandex/callback`
- Обновлена функция `verifyToken` для работы с новыми маршрутами

#### 1.3 Обновление books-service

**Измененные файлы:**
- `books-service/main.go` - Обновление подключения к базе данных
- `books-service/go.mod` - Обновление MySQL драйвера

**Ключевые изменения:**
- Добавлены таймауты для подключения к базе данных
- Настроено пул соединений
- Обновлен MySQL драйвер до версии 1.9.2

### 3. Исправление уязвимостей и багов

#### 3.1 Auth-service

**Исправленные уязвимости:**
1. **JWT библиотека:**
   - Заменена устаревшая библиотека `dgrijalva/jwt-go` на `golang-jwt/jwt/v5`

   - Обновлена функция `verifyToken` для работы с новой JWT библиотекой

#### 3.3 Books-service

## Инструкции по запуску и настройке

### 5.1 Предварительные требования

1. Установлен Go 1.24.2 или выше
2. Установлен Node.js 18+ и npm
3. Доступ к базе данных MySQL
4. Зарегистрированное приложение в Yandex OAuth

### 5.2 Настройка Yandex OAuth

1. Зарегистрируйте приложение в [Yandex OAuth](https://oauth.yandex.ru/)
2. Получите `Client ID` и `Client Secret`
3. Добавьте redirect URI: `http://localhost:8080/auth/yandex/callback`
4. Настройте переменные окружения:

```bash
# Auth-service .env
YANDEX_CLIENT_ID=ваш_client_id
YANDEX_CLIENT_SECRET=ваш_client_secret
YANDEX_REDIRECT_URL=http://localhost:8080/auth/yandex/callback

# Gateway .env
AUTH_SERVICE_URL=http://localhost:8081
BOOKS_SERVICE_URL=http://localhost:8082

# Books-service .env
DB_HOST=localhost
DB_PORT=3306
DB_USER=librarium_user
DB_PASSWORD=your_password
DB_NAME=librarium
```

### 5.3 Запуск сервисов

```bash
# 1. Запуск базы данных
docker-compose up -d mysql

# 2. Запуск auth-service
cd auth-service
go run main.go

# 3. Запуск books-service
cd books-service
go run main.go

# 4. Запуск gateway
cd gateway
go run main.go

# 5. Запуск фронтенда
cd library-reports
npm run dev
```

### 5.4 Проверка работы

1. Откройте `http://localhost:3000` в браузере
2. Перейдите на страницу входа или регистрации
3. Нажмите кнопку "Войти через Яндекс"
4. Пройдите авторизацию через Yandex
5. Убедитесь, что вы вошли в систему
